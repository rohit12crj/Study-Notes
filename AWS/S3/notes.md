✅ what is s3 tables ?
- S3 Tables refer to storing structured, queryable data in Amazon S3 using formats like Parquet and querying it with services like Athena, forming the foundation of a data lake

---
✅ pre signed urls vs cookies in s3
- Pre-signed URLs provide temporary access to a single S3 object via a signed URL, whereas signed cookies (via CloudFront) allow access to multiple resources using browser-stored authentication

✅ CORS
- CORS is a browser security mechanism that allows servers to specify which cross-origin requests are permitted using HTTP headers
